# 4d-tips-hide-tool-bar

#### 問題

アプリケーションをカスタムモードで開始すると，スプラッシュ画面とメニューバーの間に隙間ができます。

#### 回避策

ビルドを実行し，シングルユーザーアプリケーションを**ファイナライズ**すれば，ユーザーモードのツールバーは表示されないので，メニューバーのすぐ下にスプラッシュ画面が表示されるようになります。
